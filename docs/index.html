Problem Statement

The EV Charging Fragmentation Crisis

"Design solutions to enable a connected, interoperable, and reliable EV charging ecosystem that delivers a seamless and user-friendly experience across networks and platforms."

🔌

Network Fragmentation

EV drivers need separate apps for Tata Power, Zeon, ChargeZone, and Bolt — each with independent wallets and login flows.

📡

Unreliable Communication

Charging stations lose billing data during cellular outages. No standardized protocol means vendor lock-in across CPOs.

💳

Payment Friction

No unified payment system. Each network requires a separate pre-paid wallet, creating financial overhead for users.

🗺️

No Real-time Navigation

Drivers can't find available nearby chargers in real time. No route planning integration from current vehicle location.

Technical Depth

Engineering Rigor

 ISO 15118 Plug & Charge

Simulates the full 4-step cryptographic handshake: physical connection lock → TLS ECDSA certificate exchange → EMAID contract verification → OCPP TransactionEvent. No manual QR/app required.

→ SignCertificate + AuthorizeResponse + TransactionEvent

 OCPP 2.0.1 Protocol

Live JSON WebSocket log simulation with BootNotification, Heartbeat, TransactionEvent (Started/Updated/Ended), SetChargingProfile, and Reset commands rendered in real time.

→ Full message lifecycle with offline queue buffering

 Smart Grid Throttling

Simulates substation peak load at 92%. Automatically triggers OCPP SetChargingProfile to limit active chargers from 150kW → 70kW. Displays real-time "GRID THROTTLED" badge on driver console.

→ SetChargingProfile + live kW gauge update

 Dynamic Station Synthesis

Uses HTML5 Geolocation to get the driver's real coordinates, then dynamically generates IOCL, HPCL, BPCL, Jio-bp petrol-pump EV hubs around their actual location. Plots glowing route polyline to selected station.

→ GPS + dynamic CPO data synthesis

Architecture

Modular Angular 18 Architecture

📁 Standalone Components

AppComponent — Global layout, nav, theme/role switcher
DashboardComponent — SOC gauge, Plug & Charge timeline
MapComponent — Leaflet map, geolocation, routing
OcppTerminalComponent — Protocol monitor & grid controls
WalletComponent — Unified payment & session ledger
AnalyticsComponent — Chart.js SoH & CO₂ tracking

⚙️ Shared Services & State

EcosystemService — Central singleton state (walletBalance, vehicleSoc, activeSession, ocppLogs, stationsData, currentRole, currentTheme)

Key Patterns: Observable callbacks for chart refresh on session end. Role-based conditional rendering. Theme class toggling on document.body. Leaflet hidden/shown with invalidateSize().

→ Zero external backend — 100% browser-side

🚀 Deployment

Angular 18 → NGINX static serve → Render.com / Docker container. Build output: dist/voltstream/browser. Production bundle: ~378 KB.

Innovation

What Makes VoltStream Innovative

🏦

Interoperable Settlement Layer

A single pre-funded wallet automatically settles billing across 4+ competing CPO networks — like a UPI for EV charging. No per-network app or wallet required.

⚡

Live Liquid Charging Animation

Rotating glow ring + rising energy bubbles + pulsing bolt glow on the radial SOC indicator. Physics-accurate visual feedback during active charging — a first for EV dashboards.

🛢️

Petrol Pump EV Hub Synthesis

Detects real driver GPS coordinates and synthetically generates IOCL, HPCL, BPCL, Jio-bp petrol pump EV charging points around them — matching India's actual fuel network rollout strategy.

🔋

Battery SoH Digital Passport

AI-driven State of Health tracker generates battery lifecycle data qualifying aging cells for second-life grid storage — reducing lithium supply chain dependency.

🌐

Dual-Perspective Role System

Single codebase serves two completely different user experiences: EV Driver Companion (consumer UX) and Grid Operations Console (NOC operator), switching instantly with zero page reload.

🛡️

Smart Grid Protection

Simulates substation overload → OCPP SetChargingProfile → automatic throttling from 150kW → 70kW, protecting grid infrastructure in real time.
